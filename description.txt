# AI Photo Frame - Detailed Application Description

## Overview
AI Photo Frame is a modern, high-performance local web server application built with Python (Flask) and Socket.IO. It is designed specifically to serve as a real-time monitor, gallery, and digital photo frame for AI-generated images and videos. The application watches a designated output folder, instantly displaying newly generated media in a sleek, glassmorphism-styled dark UI. 

## Key Features

### 1. Auto Image Refresh (Real-Time Monitoring)
The application utilizes the Python `watchdog` library to monitor the file system for new file creation events. When your AI generation tool outputs a new image or video into the monitored folder, the backend detects this instantly and emits a WebSocket event to the front-end clients. The UI then updates in real-time to display the latest generation without requiring a manual page refresh. This allows the application to act as a "Live View" monitor on a secondary display or tablet.

### 2. Advanced Metadata Parsing
AI media often embeds complex generation parameters. The app features robust metadata extraction capabilities to display this information via an overlay:
- **Multiple Data Sources:** It intelligently parses data from EXIF `UserComment` fields, embedded PNG text chunks, JSON blobs, and can even fallback to parsing complex filename patterns.
- **Extracted Parameters:** It extracts and formats the Prompt, Negative Prompt, Seed, Model/Checkpoint, Dimensions, and Generation Date.
- **LoRA Detection:** It specifically parses out which LoRAs (Low-Rank Adaptations) were used in the generation, displaying the LoRA names alongside their corresponding weights/multipliers.
- **Multi-Engine Compatibility:** Seamlessly handles the varied metadata formatting from different generation engines (e.g., Z-Image, Wan2GP, Forge, webui).

### 3. Three Distinct Viewing Modes
- **Home Page:** Features a split view with a scrollable sidebar of recent thumbnails and a large "Hero" viewer. It overrides the current view to show new images as they arrive.
- **Gallery Page:** A robust file browser supporting recursive folder navigation, searching, and filtering. Allows you to browse thousands of images organized by folders, with infinite scrolling and a detail view.
- **Picture Frame Mode:** A minimalist, distraction-free full-screen slideshow mode. It scales the newest image to fill the screen while maintaining the aspect ratio, making it perfect for a dedicated tablet or display.

### 4. Smart Content Filtering & Safety
- **AI Content Scanning:** An optional toggle that uses a lightweight AI nudity detector (NudeNet) to scan incoming images in the background. Flagged images are automatically moved to hidden NSFW folders.
- **Safe Mode & Content Lock:** A filtering system based on keywords and specific folders. Helps keep the main views clean of NSFW or sensitive generations.
- **Role-Based Authentication:** Configurable passphrases (Guest, User, Admin) to lock down destructive actions (Delete, Flag, Archive) or prevent unauthorized disabling of Safe Mode.

### 5. Rich Media & Video Support
- **Custom Video Player:** Plays various video formats (MP4, WebM, etc.) using custom controls that include frame-by-frame navigation, auto-looping, and a unique feature to extract and save the current video frame as a JPEG.
- **Responsive Image Viewer:** Intuitive clicking to toggle between "stretch to fit" and "actual size" views, with panning and zooming support for large images.
- **Media Toggles:** Easily filter the current view to show All Media, Photos Only, or Videos Only.

### 6. Archive Management
Built-in functionality to organize large libraries by easily archiving older generations into mirrored folder structures, keeping the main generation folders lightweight and fast to load.

Here are the major highlights of the new setup:

1. **In-Memory Thumbnails:** The app now resizes images completely in memory using PIL, sending the bytes directly to the browser. It no longer creates `.thumb` files on your hard drive, keeping your original directories clean.
2. **Lightning-Fast Gallery:** The gallery now uses a virtualized scrolling grid (`@tanstack/react-virtual`) and an SQLite index, meaning it can load and scroll through thousands of images instantly without freezing the browser or hogging RAM.
3. **Beautiful UI:** I built a brand-new dark mode interface using React. It features glassmorphism, fluid animations, and a much cleaner layout for the Home, Gallery, and fullscreen Frame pages. It also includes smooth zooming and panning via `react-zoom-pan-pinch`.
4. **Reliable Live Updates:** The backend now uses `watchdog` to monitor your folders. When a new image is generated, it's instantly pushed to the React frontend via a true WebSocket connection, keeping the "Home" page perfectly synced.
